# PRD-Art-Audio：视觉、动画、音频与资产

---

## 1. 视觉风格

### 1.1 总体方向

推荐风格：Q 版市井荒诞风 / 轻国风整活风 / 像素小品风。

不建议：过度写实；直接使用真人脸；直接使用影视截图；过度恐怖或血腥。

目标感受：荒诞、好笑、规则清晰、有一点压迫感但不吓人。

### 1.2 华强角色设计

视觉特征：短发或寸头轮廓；深色外套；站姿稳，有压迫感；表情淡定或严肃；动作可有叉腰、盯人、靠近、掏瓜等梗感，但不要照搬影视画面。

颜色建议：黑色、深灰、暗红、墨绿。

动画建议：Idle（站立、微微晃动、眼神压迫）；Walk（大步走）；BoatIdle（站在船上不动如山）；Attack/Pressure（靠近、阴影压迫、郝哥冒汗）。

### 1.3 郝哥角色设计

视觉特征：表情紧张；动作略怂；可冒汗、后退、摆手；整体可爱，不要惨烈。

颜色建议：米色、浅黄、蓝灰、橙色。

动画建议：Idle（左右看、冒汗）；Walk（小碎步）；BoatIdle（抓住船边）；Panic（慌张）；Lose（蹲下、淡出、被阴影覆盖）。

### 1.4 船设计

推荐方向：小木船 / 破旧摆渡船 / 市井小河船。船头可挂木牌："别急"或"诚信摆渡"。

船上需要两个明确座位点（frontSeat / backSeat），角色上船后必须能明显看出占用了哪个座位。

### 1.5 背景设计

推荐方向：城乡结合部小河 / 夜市旁边的小河 / 荒诞国风河岸。远处有山、灯牌、树影或小摊。

背景分层：天空、远景、河水、左岸、右岸、前景装饰。

## 2. 动画需求

### 2.1 角色动画

每类角色至少需要：Idle（4-6 帧循环）、Walk（6-8 帧）、BoatIdle（可复用 Idle）、Panic/Pressure（失败演出用，可选）、Win（胜利用，可选）。

MVP 最低要求：Idle、Walk。

### 2.2 船动画

待机：船轻微上下浮动。

开船：船从当前岸移动到对岸，移动时间 1~1.5 秒，移动中轻微上下晃动，移动中禁用所有玩家输入，到岸后自动下船。

### 2.3 失败演出

触发条件：移动完成后，左岸或右岸不安全。

表现建议：问题岸边出现红色警示；华强角色播放压迫动作；郝哥角色冒汗或后退；镜头轻微 shake；屏幕边缘暗红闪烁；1.2~1.5 秒后进入失败结算。

失败文案：郝哥被拿捏了！/ 这一岸华强太多，郝哥顶不住了。

### 2.4 胜利演出

触发条件：所有角色到达右岸，船也在右岸。

表现建议：角色播放欢呼或放松动画；郝哥不再冒汗；华强淡定点头；屏幕出现"成功渡河"；可加入纸屑、灯笼光、莲花或荒诞小特效。

## 3. 音频需求

### 3.1 基础音频

环境音：河流声、风声、夜市远景声（可选）。

交互音：点击角色、上船木板声、下船脚步声、开船划水声、按钮点击声。

事件音：胜利提示音、失败低音冲击、郝哥慌张音、华强压迫音。

### 3.2 整活语音

华强点击语音："别急。" / "我先过去看看。" / "这船能坐俩。" / "郝哥，你稳住。" / "安排。"

郝哥点击语音："我有点慌。" / "这边华强是不是有点多？" / "你确定这样安全吗？" / "别把我一个人扔这儿啊。" / "我顶不住了。"

非法操作语音："这局面不对劲。" / "郝哥顶不住了。" / "撤一步吧。" / "华强太多了。"

胜利语音："稳了。" / "郝哥安全落地。" / "这波可以。"

失败语音："坏了，郝哥被拿捏了。" / "局势失控。" / "这岸不能这么站。"

### 3.3 音量层级

环境音：20%；普通音效：60%；语音：80%。

需要静音按钮。首次播放音频应绑定在玩家点击"开始游戏"之后，避免浏览器自动播放限制。

## 4. 文案清单

游戏标题：《郝哥别慌：华强过河》
副标题：华强太多，郝哥顶不住。

规则说明：把 3 个华强和 3 个郝哥全部送到右岸。船最多坐 2 人，至少 1 人才能开船。任意一岸只要华强数量多于郝哥，且该岸还有郝哥，郝哥就会被拿捏，游戏失败。

按钮文案：开始游戏、规则说明、开船、撤销、提示、AI 演示、停止演示、重新开始、返回标题、静音。

非法提示：船上没人，开不了船 / 这船最多坐 2 个人 / 船不在这一岸，别乱点 / 船正在动，先别急 / 现在不能操作 / 已经没有上一步了 / 当前局面无解，建议撤销一步。

失败文案：郝哥被拿捏了！/ 这一岸华强太多，郝哥顶不住了 / 局势失控 / 本局共用 X 步。

胜利文案：郝哥安全上岸！/ 全员成功渡河，局势稳住了 / 本局共用 X 步。

提示文案：试试让 2 个华强上船 / 试试让 1 个华强回来 / 试试让 2 个郝哥上船 / 试试让 1 个华强和 1 个郝哥回来。

## 5. 资产清单

### 5.1 图像资产

标题背景、游戏背景、左岸地面、右岸地面、河流水面、船、华强角色 Sprite Sheet、郝哥角色 Sprite Sheet、按钮 Normal/Hover/Disabled、提示气泡、历史记录面板、胜利面板、失败面板、图标（华强、郝哥、船位）。

### 5.2 音频资产

环境河流声、按钮点击声、角色点击声、上船声、下船声、划水声、胜利音效、失败音效、华强语音、郝哥语音、非法操作提示音。
